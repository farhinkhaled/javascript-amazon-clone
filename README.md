# JavaScript Amazon Clone 

This project is a **frontend Amazon-like e-commerce clone** built as part of the **SuperSimpleDev JavaScript course**.


<br>

## Features

* Product listing with dynamic rendering
* Add to cart / remove from cart functionality
* Quantity updates with real-time price calculation
* Persistent cart using `localStorage`
* Order summary with subtotal, tax, and total price calculation
* Modular JavaScript structure (separation of concerns)
* Responsive layout using HTML & CSS

<br>

## Tech Stack

* **HTML**
* **CSS**
* **JavaScript**
* **LocalStorage API**

<br>

## Project Structure

```text
├── data/          # Product and cart data
├── scripts/       # JavaScript logic (cart, checkout, products)
├── styles/        # CSS files
├── index.html     # Main product page
└── checkout.html  # Checkout page
```
